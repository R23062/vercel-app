## 概要
「いま、この瞬間の気持ち」をシンプルかつ直感的に共有できるリアルタイム掲示板アプリです。
複雑な機能を削ぎ落とし、誰でも迷わず投稿・交流ができる「ミニマルなSNS」を目指して開発しました。

### ターゲットユーザー
- 気軽に今の状況をメモしたい人
- リアルタイムで他人の反応を感じたい人
- ログインの手間なく、すぐにコミュニケーションを始めたい人

## 特徴と機能の説明

### 1. モダンな2カラムレイアウト
左側に「投稿ボード」、右側に「タイムライン」を配置。視線の移動を最小限に抑え、PCでの操作性を追求しました。

<img width="959" height="500" alt="p0" src="https://github.com/user-attachments/assets/ede75840-c1ad-493e-acd4-4e3c0208f0e7" />

### 2. リアルタイム・スレッド機能
Supabase Realtime を活用し、ページ更新なしで投稿が反映されます。また、特定の投稿に対する「返信（リプライ）」が可能で、スレッド形式で会話が繋がります。

<img width="595" height="264" alt="p1" src="https://github.com/user-attachments/assets/48714a8b-87d0-482e-8197-74c3f3abe910" />

### 3. 直感的なリアクションと管理
「いいね（❤️）」ボタンによるクイックな反応や、不要になった投稿の削除機能を搭載。ユーザーが自分の発信をコントロールしやすくしています。

名前入力欄を設けることで、匿名性を保ちつつも「誰が話しているか」をアイコンの色と文字で視覚的に判別可能にしました。

## 使用技術 (技術スタック)

### フロントエンド / バックエンド
- **TypeScript**: 静的型付けによる堅牢な開発
- **Next.js (App Router)**: 高速なレンダリングと快適なUXの実現
- **Tailwind CSS (v4)**: モダンで一貫性のあるデザインの実装
- **Supabase**: データベース管理およびリアルタイム通信の基盤

### インフラ・ツール
- **Vercel**: アプリケーションのホスティング
- **GitHub Actions**: Supabaseの停止防止（Auto-Ping）の実装
- **GitHub Codespace**: 開発エディタ兼ターミナル

### システム構成
[ユーザー] <--> [Next.js (Vercel)] <--> [Supabase (Realtime / DB)]

## 工夫した点・苦労した点
- **リアルタイム性の確保**: Supabaseの `subscribe` 機能を用い、投稿が即座に反映される心地よさを追求しました。
- **スレッド表示のロジック**: データベース上では平坦なデータを、フロントエンド側で親投稿と返信投稿を紐付け、適切な位置にインデント（ズレ）させて表示させるアルゴリズムを工夫しました。
- **環境構築**: 開発途中で環境の再構築が必要になりましたが、GitHubでのバージョン管理を徹底していたため、迅速に復旧・進化させることができました。

## 既知の課題と今後の展望
- **画像投稿機能**: 現在はテキストのみですが、画像の添付機能を追加したいと考えています。
- **検索・フィルタリング**: 投稿数が増えた際、特定のキーワードで検索できる機能の実装。
